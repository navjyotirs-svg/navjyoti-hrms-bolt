# Navjyoti HRMS — Bolt.new Start Here

## Decision
Build a new full-stack Bolt project using the current HTML only as a visual/workflow reference.

## Out of scope
Payroll, salary, payslips, compensation, incentives, salary deductions, statutory payroll and performance-linked financial calculations.

## Steps
1. Create a new Bolt project.
2. Connect GitHub.
3. Add all Markdown files from this pack to the project root.
4. Keep the legacy HTML only under `/reference`.
5. Paste `BOLT_MASTER_PROMPT.md` into Bolt.
6. Complete one phase at a time.
7. Save a version-history checkpoint after every accepted phase.

The legacy HTML is not production code because it uses mock users, browser-side persistence, browser timestamps and client-only reminders.
