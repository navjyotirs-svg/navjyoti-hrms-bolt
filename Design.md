# Navjyoti HRMS — Design

Preserve the legacy prototype’s professional ink/paper/teal visual character while rebuilding responsive React components.

Tokens:
- Ink #14181F
- Ink secondary #1D2330
- Paper #F7F5F0
- Surface #FFFFFF
- Border #E4E0D6
- Teal #0E6E63
- Teal dark #0A544B
- Amber #C17817
- Rose #B3413A
- Slate #5B6472

Typography: Zilla Slab headings, Inter UI/body, IBM Plex Mono IDs/times.

Required UI: responsive navigation, KPI cards, task history, attendance card/countdown, camera checkout modal, location state, leave cards, calendar, tickets, daily report editor, notification inbox, high-priority reminder modal, audit timeline and complete loading/empty/error/permission states.

Do not create payroll/salary/incentive/deduction components. Attendance must not show late minutes or salary impact. Target WCAG 2.1 AA and 44px touch targets.
